# 서울식물원 (리뉴얼)

## :computer: 프로젝트 소개
### 선정이유<br>
기존 웹 페이지는 복잡하고 중복되는 카테고리 구성 및 디자인과서울식물원의 콘텐츠를 쉽게 볼 수 없어<br>
사용자가 이용하기에 불친절한 페이지라고 판단하였습니다.

### 목표<br>
해당 페이지의 카테고리 구성 간소화 및 사용자들이 콘텐츠를쉽게 찾아볼 수 있도록 하는 것에 중점을 맞추고<br>
디자인과 기능을 개선하고자 하였습니다.
